Toronto Airbnb Pricing Prediction Project

This project is intended to explore the dataset from Toronto Airbnb Website, that records Airbnb listing prices along with other features for the property. The record date of the data is Feb 14, 2020. 

The full dataset can be retrieved here:
http://insideairbnb.com/get-the-data.html

The project tries to identify top features that have an effect on the final listing price of a property, and uses Machine Learning techniques to build and compare different models that can predict the final listing price of a property, give all other relevant features.
